SCREENSHOT REQUIREMENTS:
- Minimal: 3 screenshots
- Maksimal: 8 screenshots  
- Resolusi minimal: 320x640px
- Aspect ratio: 1:2 (portrait mode)
- Format: PNG
- Recommended: 1080x1920px atau 1440x2560px

CARA MENGAMBIL SCREENSHOTS:
1. Run aplikasi di emulator atau device
2. Gunakan Android Studio screenshot tool atau
3. Gunakan ADB: adb shell screencap -p /sdcard/screenshot.png
4. Atau gunakan device screenshot langsung

TIPS UNTUK SCREENSHOT YANG BAIK:
- Pastikan semua text jelas terbaca
- Tidak ada loading spinner (kecuali untuk splash)
- UI elements tidak terpotong
- Warna kontras yang baik
- Menunjukkan key features aplikasi
- Fokus pada compliance features (contact, aggregator declaration)

Simpan dengan nama yang descriptive dan urutkan sesuai flow user experience.
